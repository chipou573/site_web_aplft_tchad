
# APLFT Website

Ce site est conçu pour représenter l'association APLFT, basée sur la maquette fournie.

## Contenu

- `index.html` : Page principale du site.
- `styles.css` : Feuille de style CSS.
- `logo.png`, `action1.jpg`, etc. : Images du site (à ajouter dans le même dossier).
- `map.png` : Carte de Diameria.

## Instructions

1. Placez toutes les images dans le même dossier que `index.html`.
2. Ouvrez `index.html` dans un navigateur pour voir le site.
3. Hébergez le dossier sur un serveur ou via GitHub Pages pour mise en ligne.
